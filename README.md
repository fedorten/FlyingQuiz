# Порящий Квиз (FlyingQuiz)

Игра на реакцию и внимательность. На экране появляются летящие слова, нужно кликать на целевое слово, которое отображается вверху экрана.

## Как играть

1. Открой `index.html` в браузере (через HTTP сервер, не как файл)
2. Нажми "Играть"
3. Кликай на слова, совпадающие с целевым словом вверху экрана
4. За каждое угаданное слово +100 очков, за неправильное -10
5. Задача - набрать максимум очков за отведённое время

## Запуск

### Локально

```bash
# Python
python -m http.server 8000

# Node.js
npx serve

# Затем открыть http://localhost:8000
```

### GitHub Pages

1. Создать репозиторий
2. Загрузить файлы: index.html, config.js, words.txt
3. Settings → Pages → выбрать branch
4. Игра доступна по URL

## Конфигурация (config.js)

| Параметр | Описание | Значение |
|----------|----------|----------|
| gameDuration | Длительность игры (мс) | 30000 |
| maxWordsOnScreen | Максимум слов на экране | 50 |
| wordSpeed | Скорость движения слов | 7 |
| wordSpeedVar | Вариация скорости (+/-) | 3 |
| spawnInterval | Интервал появления слов (мс) | 500 |
| spawnRate | Множитель скорости спавна | 1 |
| pointsCorrect | Очки за правильное слово | 100 |
| pointsWrong | Штраф за неправильное | -10 |
| targetWordChance | Шанс что летящее слово будет целевым | 0.15 |
| targetFromScreen | Выбирать целевое слово с экрана | true |
| targetFromScreenChance | Шанс выбора с экрана (0-1) | 1 |

## Архитектура кода

### Переменные

- `words` - массив всех слов из words.txt
- `targetWord` - текущее целевое слово
- `usedTargetWords` - Set использованных слов (чтобы не повторялись)
- `activeWords` - массив объектов {element, x, y, vx, vy, text} слов на экране
- `score` - текущий счёт

### Функции

- `loadWords()` - загрузка слов из words.txt
- `getRandomWord(excludeWords)` - получить случайное слово (с опциональным исключением)
- `setNewTarget()` - установить новое целевое слово
- `createWord(forceTarget)` - создать летящее слово
- `handleWordClick(wordObj)` - обработка клика по слову
- `updateWords()` - обновление позиций слов (вызывается каждые 16мс)
- `startGame()` - начало игры
- `endGame()` - окончание игры

### Логика выбора целевого слова

1. Если `targetFromScreen: true` + на экране ≥2 слов + выпал шанс `targetFromScreenChance`:
   - Выбирается слово с экрана (исключая уже использованные)
2. Иначе:
   - Случайное слово из списка (исключая использованные)
